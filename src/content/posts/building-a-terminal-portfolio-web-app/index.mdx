    legible to non-terminal users.
  - TanStack on the client and Elysia on Bun on the server keep command flows
    fast and structured.
  - SQLite fits a content-and-command app well when the data model stays local,
    simple, and queryable.
  - Deployment architecture matters because low-latency command UX falls apart
    if the stack feels remote or heavy.
